### 1. Метод `run()` - главный цикл агента

**Файл**: `agent/core.py:~400`

**Сигнатура**:
```python
async def run(
    self,
    task: str,
    workspace_path: str | Path,
    plan_mode: str | None = None
) -> AsyncIterator[dict]:
```

**Фазы**:
1. **Observe** - сканирование workspace
2. **Recall** - поиск в памяти
3. **Reason** - генерация кода через LLM

5. **Commit** - сохранение в память
